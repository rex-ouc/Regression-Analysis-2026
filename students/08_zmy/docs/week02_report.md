## 1. 实验目的
- 生成一元线性回归数据：\( y = 1 + 2x + \epsilon \)，\(\epsilon \sim N(0,1)\)。
- 使用最小二乘法公式手动计算参数 \(\hat{\beta}_0\) 和 \(\hat{\beta}_1\)。
- 计算 \(\hat{\beta}_1\) 的方差及估计偏差。
- 调用 sklearn 和 statsmodels 验证手动结果。

## 2. 数据生成
设置随机种子 42，生成 100 个样本。\(X\) 在 \([0,10]\) 上均匀分布，误差项服从标准正态分布。

## 3. 结果

### 3.1 手动计算结果
- \(\hat{\beta}_0 = 1.215096\)
- \(\hat{\beta}_1 = 1.954023\)
- \(\mathrm{Var}(\hat{\beta}_1) = 0.00093939\)
- 偏差：\(\beta_0\) 偏差 = +0.215096，\(\beta_1\) 偏差 = -0.045977

### 3.2 方法对比
| 方法               | \(\hat{\beta}_0\) | \(\hat{\beta}_1\) |
|--------------------|-------------------|-------------------|
| 真实值             | 1.000000          | 2.000000          |
| 手动公式           | 1.215096          | 1.954023          |
| sklearn            | 1.215096          | 1.954023          |
| statsmodels(公式)  | 1.215096          | 1.954023          |
| statsmodels(数组)  | 1.215096          | 1.954023          |

## 4. 结论
手动计算结果与 sklearn、statsmodels 完全一致，验证了公式的正确性。偏差较小，符合统计理论。
